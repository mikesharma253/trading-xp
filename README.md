# trading-xp
complete trading simulator game in one file using pygame. It has:  Live moving price chart 📊 Buy / Sell system 💰 Balance + holdings tracking Profit calculation Random market + trend News events (price spikes/drops) 📰 Clean UI + controls
Live candlestick charts with multiple timeframes (1m, 5m, 15m, 1h, etc.)
Different chart types: candlestick, line, and bar
Drawing tools like trendlines, Fibonacci, rectangles, and horizontal levels
Buy and sell system with real-time profit and loss tracking
Basic technical signal indicator based on moving averages and trend
Moving averages (20 and 50)
Volume panel
Simulated live news that influences market direction
Watchlist with multiple symbols
Positions panel to manage trades
Account dashboard showing balance, P&L, win rate, and more
Keyboard shortcuts for faster interaction
This is a paper trading simulator, which means no real money is involved. Prices are generated using a simulation engine, and market behavior is influenced by random movement and news events.
